THERE ARE MANY DIFFERENT CLASSES OF PEOPLE THAT SAILED ON THE TITANIC



COST

1st class tickets cost $2,500 in todays money that would cost $103,000.

2nd class tickets cost $65 in todays money that would cost $1,159.

3rd class tickets cost $36 in todays money that would cost $642.







CLASSES

There were 3 classes. Most of the passengers were 3rd class







FACILITIES
